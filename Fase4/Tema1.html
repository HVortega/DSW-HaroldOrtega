LAS ETIQUETAS SEMÁNTICAS - HTML-5

Sirven para definir la estructura del documento en los sitios web modernos.

Las Etiquetas Semánticas
  • HEADER: La cabecera de una página, se usa para marcar un grupo de elementos de introducción o de navegación dentro de una sección o documento. Normalmente se usa para incluir los encabezados (h1) (h6) o (hgroup) pero no es obligatorio. También puede contener otras cosas como el índice de una sección, un formulario de búsqueda, logos relevantes.

  • NAV: Con el que colocar al navegador los menús principales de una página web.

  • SECTION: Es una sección dentro de un documento. Se define un área de contenido única dentro del sitio.

  • ARTICLE: Especifica un artículo, es decir, una unidad de contenido.

  • FOOTER: El pie de página o informaciones que formen el pie de una sección.

  • ASIDE: Es una parte de la web que muestra contenido accesorio, generalmente se coloca en un panel lateral. La etiqueta se usa para marcar un trozo de contenido que está relacionado con el contenido de la página web, pero que no es parte del mismo. Ejemplos de uso serían: glosario de términos, grupos de enlaces a páginas relacionadas, barras laterales, entre otros.
